# Cove Website

The Cove website supports the desktop runtime. It is not an AI chat interface, an agent dashboard, or the primary product.

## Website Purpose

- Downloading Cove
- Account management
- Device management
- Cloud sync
- Security settings
- Documentation

## Pages

- Landing page
- Download page
- Account page
- Device management page
- Security page

## Local Development

```bash
npm install
npm run dev
```
