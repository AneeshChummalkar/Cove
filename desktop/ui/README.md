# UI

Status: architecture placeholder. No production code belongs here during Phase 3.

## Purpose

The desktop UI provides Cove's visible presence, trust dashboard, activity log, permission controls, memory controls, and platform-specific surfaces.

## macOS Surfaces

- Notch UI
- Menu bar access
- Cursor popup
- Floating assistant
- Contextual notifications

## Windows Surfaces

- System tray
- Floating orb
- Cursor popup
- Contextual notifications

## Required Views

- Permission dashboard
- Activity log
- Active agents
- Memory inspector
- Trusted devices
- Security controls
- Kill switch state

## Trust Rule

The UI must always make it clear what Cove sees, remembers, controls, and is doing now.
