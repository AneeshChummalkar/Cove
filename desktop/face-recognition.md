# Face Recognition

Face recognition is an optional future identity layer.

## Uses

- Confirm user presence
- Increase confidence for sensitive actions
- Lock Cove when the user leaves
- Detect unknown users during sensitive requests

## Requirements

- Explicit opt-in
- Local-first processing when practical
- Clear camera indicator
- No silent enrollment
- Easy disable and delete controls
- No irreversible action based only on face confidence

## Privacy

Face templates should be stored locally by default and protected with OS secure storage or platform biometric APIs where available.
